
# 🌦️ Weather Dashboard

A simple and interactive Weather Dashboard built with **Python**, **Streamlit**, and the **OpenWeatherMap API**. Users can search for any city and view real-time weather information.

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Current temperature
- 💧 Humidity
- 🌬️ Wind speed
- ☁️ Weather condition
- 🌅 Sunrise and sunset time
- 🏳️ Country information
- 🖼️ Weather icon
- ⏰ Last updated time

## 🛠️ Technologies Used

- Python
- Streamlit
- Requests
- OpenWeatherMap API
